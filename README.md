# 2048 Game Webapp

A fully functional and responsive 2048 game built with HTML, CSS, and JavaScript. This webapp works seamlessly across all devices including desktops, laptops, tablets, and mobile phones.

## 🎮 Features

- **Classic 2048 Gameplay**: Slide tiles to combine same numbers and reach 2048
- **Responsive Design**: Optimized for all screen sizes and devices
- **Touch Support**: Swipe gestures for mobile devices
- **Keyboard Controls**: Arrow keys for desktop users
- **Score Tracking**: Current score and best score persistence
- **Undo Function**: Go back one move if you make a mistake
- **Beautiful UI**: Modern gradient design with smooth animations
- **Local Storage**: Best score saved in browser
- **Game Over Detection**: Automatic game over screen when no moves are possible

## 🚀 How to Play

1. **Objective**: Combine tiles with the same number to reach 2048
2. **Controls**:
   - **Desktop**: Use arrow keys (↑↓←→)
   - **Mobile**: Swipe in any direction
   - **Touch Buttons**: Use on-screen arrow buttons on mobile
3. **Gameplay**: 
   - Tiles slide in the direction you choose
   - Same numbers merge when they collide
   - A new tile (2 or 4) appears after each move
   - Game ends when no more moves are possible

## 🎯 Game Rules

- Start with two tiles (2 or 4)
- Slide tiles in any direction
- When two tiles with the same number collide, they merge into one
- After each move, a new tile appears randomly
- Goal is to create a tile with the number 2048
- Game continues until no more moves are possible

## 🏆 Tips for High Scores

1. **Corner Strategy**: Keep your highest tile in a corner
2. **Snake Pattern**: Build a snake-like pattern of decreasing numbers
3. **Avoid Scattered Tiles**: Keep similar numbers close together
4. **Plan Ahead**: Think 2-3 moves ahead
5. **Use Undo**: Don't hesitate to undo a bad move
